# multiplo-mayer
O Multiplo de Mayer é um indicador de mercado criado por Trace Mayer, que divide o preço atual do ativo pela sua média móvel de 200 dias. Ele é usado para analisar se o preço do ativo está sobrevalorizado ou subvalorizado.
